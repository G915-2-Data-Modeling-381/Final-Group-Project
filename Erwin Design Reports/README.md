## Erwin Design Reports

Follow proper naming conventions for design reports as follows: 
- '**G915_2**-**NameOfErwinFile**-**V#**-**TypeOfReport**-**Initials**-**YYYYMMDD**.documenttype'
  -  NameOfErwinFile: Name of the file and version number (V#)
  -  TypeOfReport: If you used a template put the name of it here, otherwise some of the main option toggles you used
  -  Initials: Your own initials
  -  YYYYMMDD: The current date
